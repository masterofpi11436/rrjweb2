# Consoldated Project

Contains all prjects created for RRJ

Contains the following projects:

# Phone Directory
Phone extensions for all the phones

# Employee Contacts
Employee contact information for Master Control

# Inmate Tablet Tracker 
Keep track of inmates that have broken, destroyed, and lost tablets for ICS

# Mailroom List
List of Inmates that need to have their mail forwarded to OPR

# Names
A search engine that searches names of inmates with Restricted tablet access for ICS and Needs mail forwarded to OPR to sift through the electron mail

# Admin
Dashboard for the IT Administrator to control who has access to what program

# Warehouse
Warehouse Supply request is used to allow users to submit requests to the warehouse to get supplies
